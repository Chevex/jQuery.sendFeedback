jQuery.sendFeedback Demo
This is a demo of the jQuery.sendFeedback plugin.

The plugin is still largely a work in progress but it is moving quickly. If you would like to follow or contribute to the project then be sure to visit the project page on Google Project Hosting. http://code.google.com/p/sendfeedback/

To begin the demonstration simply click the "Send Feedback" button in the lower right corner.

This demo also uses a jQuery plugin called "colorbox". It displays a "thank you" colorbox after feedback has been sent by handling the "feedbackSent" callback function when making the call to the plugin. Colorbox is not a requirement for jQuery.sendFeedback, it is only used as an example of something you could do in a callback for the purposes of this demonstration.
by Alex Ford at CodeTunnel.com.
